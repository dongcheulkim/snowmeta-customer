## 🎯 1단계: 호스팅 서비스 선택

### ⭐ 추천: Railway.app (가장 쉬움)

**장점:**
- GitHub 연동만으로 자동 배포
- 무료 도메인 제공 (snowmeta.railway.app)
- SSL 인증서 자동 적용
- 데이터베이스 포함
- 클릭 몇 번으로 배포 완료

**가격:** 월 $5 (약 6,500원)

**배포 방법:**
1. https://railway.app 회원가입
2. GitHub 계정 연동
3. "New Project" → "Deploy from GitHub repo"
4. 이 프로젝트 선택
5. 환경변수 설정 (아래 참고)
6. 배포 완료! 🎉

### 대안: Cafe24 호스팅 (한국어 지원)

**장점:**
- 한국어 고객센터
- 전화 상담 가능
- 안정적인 국내 서버

**가격:** 월 10,000원~

**사이트:** https://www.cafe24.com

## ⚙️ 3단계: 환경 변수 설정

호스팅 서비스에서 다음 환경 변수를 설정하세요:

```env
NODE_ENV=production
PORT=5006
CORS_ORIGIN=*
DB_HOST=localhost
DB_USER=postgres
DB_PASSWORD=SnowMeta2024!@#SecurePassword
DB_NAME=snowmeta_customers
DB_PORT=5432
JWT_SECRET=SnowMeta2024JWTSecretKey!@#$%^&*()
SESSION_SECRET=SnowMetaSessionSecret2024!@#
ENABLE_HTTPS=false
```

## 📦 4단계: 파일 업로드

### Railway.app 사용 시

### 접속이 안 돼요
- 방화벽 확인

- 서버 로그 확인

- 데이터베이스 charset 확인
